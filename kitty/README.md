# ⌨️ Raccourcis Clavier Kitty

Guide des raccourcis pour Kitty Terminal.

---

## 📑 Gestion des Onglets (Tabs)

| Raccourci | Action |
|-----------|--------|
| `Ctrl + Shift + T` | Nouveau tab |
| `Ctrl + Shift + Q` | Fermer le tab actuel |
| `Ctrl + Shift + →` | Tab suivant |
| `Ctrl + Shift + ←` | Tab précédent |
| `Ctrl + Shift + 1-9` | Aller au tab N (1, 2, 3...) |

---

## 🪟 Gestion des Fenêtres

| Raccourci | Action |
|-----------|--------|
| `Ctrl + Shift + Enter` | Nouvelle fenêtre |
| `Ctrl + Shift + W` | Fermer la fenêtre actuelle |

---

## 📋 Copier/Coller

| Raccourci | Action |
|-----------|--------|
| `Ctrl + Shift + C` | Copier la sélection |
| `Ctrl + Shift + V` | Coller |

---

## 🔍 Zoom

| Raccourci | Action |
|-----------|--------|
| `Ctrl + Shift + =` | Zoom + (agrandir la police) |
| `Ctrl + Shift + -` | Zoom - (réduire la police) |
| `Ctrl + Shift + Backspace` | Reset zoom (taille par défaut) |

---

## ⚙️ Configuration

| Raccourci | Action |
|-----------|--------|
| `Ctrl + Shift + F5` | Recharger la configuration |

---

## 💡 Tips

- **Sélection à la souris** : Copie automatiquement dans le presse-papier
- **Clic molette** : Colle le contenu du presse-papier
- **Scroll** : Molette de la souris ou `Shift + PageUp/PageDown`

---

**Configuration :** `~/.config/kitty/kitty.conf`  
**Dotfiles :** `~/dotfiles/kitty/`
